A mini app is a small self contained html file. look at `examples/` folder for some examples.

The mini-app should a single file .html file

add it to the folder of the user in the `mini-apps` folder, you can find this from the .env file. If this file do not exist run the `setup-mini-app-repo` skill to create it.

For the design use the `impeccable` skill.

When creating a mini app use mock data for the content.
